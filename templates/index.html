Format Stereo Pairs

  1. Images to process should be Side-by-Side stereo pairs with no borders, in jpg format. These can be output directly from Stereo Photo Maker.
  2. Select the device on which you wish to view your images.
  3. Upload a single jpg file or a zip file with an entire batch of images. The Zip file should include only jpg files, with no folder structure.
  4. The tool will process your image(s) and then prompt you with a download of the formatted file(s). Make sure that you have any popup blockers or ad blockers turned off for this page.
